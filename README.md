# Climate Visibility Prediction System

This project predicts climate visibility using machine learning to help farmers
make better decisions during fog and low-visibility conditions.

## Features

- Machine learning based visibility prediction
- GPS and manual location support
- Forecast-based time windows
- Mobile-friendly web interface

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python src/backend/app.py
